# 象数功效查询

根据用户描述的症状，查询《八卦象数疗法大全》中对应的象数配方。数据已预解析打包，无需 PDF 文件。

## 路径配置

```
SKILL_DIR=$HOME/.claude/skills/xiangshu
CLI=$SKILL_DIR/xiangshu_cli.py
```

## 工作流

### Step 1：提取关键词

从用户输入中提取疾病/症状关键词，去掉语气词和无关修饰。

示例：
- "我最近失眠，而且头痛得厉害" → `失眠 头痛`
- "坐骨神经痛怎么办" → `坐骨神经痛`
- "查 650 3820 是什么功效" → 改用 number 模式
- "最近高血压，还有点头晕" → `高血压 头晕`

如果用户直接提供了象数编号（如 `650·3820`、`30 80`），跳到 Step 3 用 number 模式。

### Step 2：执行症状搜索

```bash
python3 $HOME/.claude/skills/xiangshu/xiangshu_cli.py symptom <关键词> 2>/dev/null
```

若关键词较多（>2个），先用全部关键词搜索；结果 <3 条时依次减少关键词重试。

### Step 3（可选）：按象数精确查询

```bash
python3 $HOME/.claude/skills/xiangshu/xiangshu_cli.py number "<象数>" 2>/dev/null
```

### Step 4：格式化输出

将 JSON 结果整理为易读格式（使用 `matched_symptoms` 字段展示匹配症状）：

```
## 象数配方推荐

根据症状「{关键词}」，共找到 {total} 个相关配方，以下是最相关的结果：

---
### 配方一：{象数}（念法：{念法}）
**主治**：
- 匹配症状1
- 匹配症状2

**备注**：{notes，如有}

**出处**：第 X、Y 页

---
### 配方二：{象数}
...

---
> 💡 使用说明：默念象数时中速，数字之间圆点处停顿一个音节，每次至少念 20 分钟。
> 建议选 1-2 个试念，选感觉舒适、无不适反应的持续念。
```

**注意**：JSON 中 `matched_symptoms` 是仅与查询词相关的症状子集；`notes` 是从书中提取的效果说明（如"念几分钟见效"），若存在则展示。

## 象数念法规则

- 数字 1-8 读本音（一、二、三…八），0 读"零"
- `·` 不读，停顿一个音节的时间
- 例：`650·3820` 读作：六五零（停）三八二零

## 注意事项

- 返回结果按相关度排序，优先展示症状命中次数最多的配方
- 若结果 >10 条，只展示前 8 条，并告知总数
- 若结果为 0，提示用户换一个更简短的关键词重试
- **不做医疗诊断**，仅提供参考信息，提醒用户配合专业医疗

## 示例对话

用户："我最近睡眠不好，经常头痛"
→ 搜索关键词：`失眠 头痛`
→ 展示 top 5 配方，格式化输出

用户："350·820 是治什么的"
→ 用 number 模式查询 `350 820`
→ 展示该象数的主治症状列表
